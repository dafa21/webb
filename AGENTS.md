# Custom Design Agent Instructions

When building, updating, or styling interfaces in this application, you MUST always apply high-quality design principles.

1. **Design Recipes Overview**: Rely heavily on the `frontend-design` skill in your internal system skills (`/skills/system_skills/design_guidelines/SKILL.md`).
2. **Target Aesthetics**: Use specific recipes based on the feature:
    - **Mindful / Healing**: Use *Recipe 7: Atmospheric / Immersive Media* (deep backgrounds, blur filters, glassmorphism, serif text).
    - **General Dashboard/Utility**: Use *Recipe 8: Clean Utility / Minimal*.
    - **Content/Articles**: Use *Recipe 6: Warm Organic / Cultural* or *Recipe 2: Editorial*.
3. **Core Rules of Thumb**:
    - Never use random off-the-shelf gradients (e.g. generic purple to blue).
    - Always select fonts with purpose (e.g., `Playfair Display` or `Cormorant Garamond` for elegant/organic serif, `Inter` for clean data).
    - Establish a proper layout using spacing and rhythmic padding. Avoid equal spacing everywhere.

Follow these rules proactively in all future generation steps.
